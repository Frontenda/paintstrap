Theme for Kickstrap

How to use this:
1. Unzip the archive.
2. Copy the contents to Kickstrap "themes" directory.
3. In "theme.less", import this theme.
(Please refer to Kickstrap Documentation)

Color sense is unnecessary

You can paint Twitter Bootstrap in great color schemes which is posted to Adobe kuler / COLOURlovers. It is not necessary for yourself to make colors.

Easy to install

It is very simple to install to your site. Download generated CSS files and replace it with a standard one. It is also usable as a theme of Kickstrap.

Customizeable

LESS files is also generated with CSS files. Theme colors is set into variables, so it is convenient to adjustment.
